# ddrescueview_install<br>
Easy install of the dd rescue view for the Raspberry Pi.
<br>
# Install<br>
sudo apt-get install -y git<br>
git https://github.com/VeggieVampire/ddrescueview_install<br>
cd ddrescueview_install<br>
./install.sh<br>
